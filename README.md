# 🍎 Fruit Quality Detection

## Description
Fruit Quality Detection is a Deep Learning project that classifies fruits as **Fresh** or **Rotten** from uploaded images using a trained TensorFlow model. The application is built with Streamlit for an easy-to-use interface.

## Technologies Used
- Python
- TensorFlow
- Streamlit
- NumPy
- Pillow (PIL)

## How to Run

1. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

2. Run the project:
   ```
   python -m streamlit run fr.py
   ```
